## &lt;buttress-db&gt;

### Version: 2.1.4

A set of elements enabling realtime data handling from ButtressJS

Sample use:

```html
<buttress-db
  endpoint="endpoint.example.com",
  app-id="[[publicAppId]]",
  user-id="[[AuthedUserId]]",
  token="[[AuthedUserToken]]",
  loaded="{{dbLoaded}}",
  db="{{db}}">
</buttress-db>
```
